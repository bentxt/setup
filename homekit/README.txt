Dotfiles

basic dotfiles that set most important environt variable and configures shells


Filetypes:

- myfilerc    => A file that is linked to ~/.myfilerc

- mydir.l     => A folder that is linked to ~/.mydir.l

- myaltdir.d  > For this folder a folder is created undeer ~/.myaltdir


- fish-config.d  => A folder is created under ~/.config/fish
                 => The file contents are linked into ~/.config/fish/
